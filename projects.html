A ciência de hoje é a tecnologia de amanhã!

    • A Ciência de Dados se desenvolve cada vez mais ao londo dos anos. Linguagens de programção como R e Python permitem o acesso a dados de forma prática e com alta precisão, e possuem um grande número de pacotes e módulos para trabalhar estes dados. O objetivo deste projeto é construir algoritmos para buscar dados de preço de mercado em sites especfícos. Posteriomente será realizada uma análise com diversos algoritmos clássicos e inteligente de regressão, classificação e agrupamento. "Você pode acompanhar o desenvolvimento deste projeto no meu repositório do"

    • O manejo florestal é de fundamental importância para o desenvolvimento sustentável e econômico de uma floresta. Atualmente, os eforços se concentram no desenvolvimento de modelos mais eficientes. A adição de variáveis ambientais nestes modelos é uma solução interessante. Um outro viés, é o uso de técnicas da computação inteligente. A combinação desses métodos pode gerar uma importante ferramenta para estimar o crecimento e o volume de espécies florestais.
